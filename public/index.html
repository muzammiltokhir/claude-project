🚀 Firebase Functions API

Welcome to the Firebase Cloud Functions API with Express.js. This API provides both public and private endpoints with different authentication methods.

📡 Available Endpoints

GET /

Health check endpoint - Returns API status and timestamp

GET /docs

Interactive Swagger/OpenAPI documentation

GET /api/public

Public endpoint requiring company access code
🔑 Requires: x-access-code header

POST /api/public/data

Submit data to public endpoint
🔑 Requires: x-access-code header

GET /api/private

Private endpoint for authenticated Firebase users
🔑 Requires: Authorization Bearer token

GET /api/private/profile

User profile endpoint for authenticated Firebase users
🔑 Requires: Authorization Bearer token
